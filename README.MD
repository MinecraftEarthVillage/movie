# 伪哔哩哔哩 - 被删视频庇护所

> **全世界被平台方迫害者，这里是你们精心制作的庇护所**

## 🛡️ 项目使命

这是一个为抵抗平台内容审查而建立的视频庇护所。我们收录在B站等平台被无故删除、下架或封禁的优质视频内容，让创作者的用心之作不再因平台规则变动而消失。

## 🌐 立即访问
**网站链接：** https://globalvillage.广东/movie

## 📮 投稿通道
如果你的视频遭遇不公对待，欢迎投稿！

**投稿方式：** 
1. 前往 [反馈区](issues)
2. 点击"New issue"创建新的投稿申请
3. 按格式提供视频信息、被删原因（可选）、视频文件（太大的可提供网盘链接、对象存储直链等）

## 📋 收录标准
- ✅ B站或其他平台被删除的原创内容
- ✅ 无违法信息的优质视频  
- ✅ 创作者拥有版权的内容
- ❌ 涉及违法、暴力、色情内容
- ❌ 侵犯他人版权的内容

## 🏗️ 技术特色
- 纯前端架构，无需后端支持
- 仿B站UI设计，熟悉易用
- 视频数据通过JSON文件管理
- 使用Vue.js构建，轻量快速
